Como crear un proyecto con yarn en entorno de vite (Pasos)
1) npm i -g yarn //Instala yarn de manera golbal
2) yarn create vite //Creamos el proyecto y le damos un nombre
3) cd [Nombre del archivo] //Ingrese al proyecto creado
4) yarn //Escribimos yarn para instalar los archivos faltantes
5) yarn dev //Iniciamos nuestro poryecto
